# Routine Planner Pro
An advanced Android application for managing daily routines, class schedules, and custom events.

## Features
- Multiple Routine Types (Daily, Class, Custom)
- Smart Reminders & Alarms (Exact alarms, snooze options)
- Unified View & Popup Animations
- AI Generator (OpenRouter integration)
- Day/Night Themes & Material 3 UI

## Build & Release
1. Open the folder in Android Studio.
2. Wait for Gradle sync and wrapper generation.
3. Push to GitHub and tag a release to trigger the GitHub Action!
